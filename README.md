# Blog Cloudinary API

This is a simple blog API built with Node.js, Express, MongoDB Atlas, and Cloudinary for image uploads. You can create users, make posts, and upload images to Cloudinary.

---

## Features

- User authentication with JWT
- CRUD operations for blog posts
- Image uploads using Cloudinary
- MongoDB Atlas as the database
- RESTful API design

---

## Technologies Used

- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- Cloudinary
- Multer
- JWT for authentication
- dotenv for environment variables

---

## Getting Started

### Prerequisites

- Node.js installed on your machine
- MongoDB Atlas account and cluster created
- Cloudinary account created
- Postman for API testing
